# Clothing Store Web Application

This is a simple Clothing Store web application developed as a school project.  
The project uses PHP and MySQL and runs on a local server environment using WampServer.

---

## Requirements

- WampServer64
- MySQL
- Web browser (Chrome, Firefox, Edge, etc.)

---

## Setup Instructions

### 1. Database Setup

1. Open **phpMyAdmin**.
2. Create a database named: clothing_store
3. Import the provided SQL file:


This will create all required tables and data.

---

### 2. Start WampServer

1. Launch **WampServer64**.
2. Make sure Apache and MySQL services are running (green icon).

---

### 3. Project Placement

1. Copy the project folder into the WampServer `www` directory: wamp64/www/clothing_store


---

### 4. Run the Application

Open your browser and go to: http://localhost/clothing_store/frontend/


The application should now be running.
